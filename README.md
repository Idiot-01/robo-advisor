# robo-advisor

A webpage showing the mNAV fluctuation over time (updated daily) for MSTR, the flagship DATCO. Daily Bitcoin valuation is pulled with demo CoinGecko API, and MSTR market cap pulled from yahoo finance with yfinance Python package.

Historical mNAV data is taken from [bitcointreasuries.net](https://bitcointreasuries.net/public-companies/strategy).
